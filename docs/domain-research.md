# 전세 사기 예방 체크리스트 앱 — 도메인 리서치

## Context

해커톤 주제로 "전세 사기 예방 체크리스트" 앱을 선정. 개발 전 도메인 지식을 체계적으로 정리하여, 앱의 핵심 기능과 리스크 판단 로직의 근거를 확보한다.

---

## 1. 전세 사기 주요 유형 (7가지)

### 1-1. 깡통전세
- **정의**: 전세보증금 + 선순위 근저당 > 매매가
- **판단**: `(근저당 채권최고액 + 전세보증금) / 매매시세 > 80%`이면 위험
- **빈발**: 신축빌라, 부동산 하락기, 시세 불투명한 지역

### 1-2. 이중계약
- **수법**: 같은 집을 여러 세입자에게 동시 계약. 전입신고 전 시간차 악용
- **예방**: 전입세대 확인서 열람, 실거주자 직접 확인

### 1-3. 가짜 임대인
- **수법**: 위조 신분증/인감증명서로 소유자 사칭
- **예방**: 등기부등본 소유자 ↔ 신분증 대조, 본인서명사실확인서 요구

### 1-4. 근저당 과다 + 경매
- **수법**: 계약 후 추가 근저당 설정 → 의도적 연체 → 경매 → 보증금 미반환
- **예방**: 특약("추가 담보 설정 금지"), 정기 등기부등본 모니터링

### 1-5. 대리인/위임장 사기
- **수법**: 위조 위임장으로 대리 계약
- **예방**: 공증 위임장 + 인감증명서(3개월 내) + 소유자 직접 전화/영상 확인

### 1-6. 신탁등기 악용
- **수법**: 신탁된 부동산을 위탁자가 수탁자(신탁회사) 동의 없이 임대 → 계약 무효
- **예방**: 등기부등본 갑구 "신탁" 확인 → 신탁원부 열람 → 신탁회사 임대 동의서 확보

### 1-7. 신종 수법 (2023~2026)
- 빌라왕(갭투자 연쇄 부도), 법인 명의 사기(페이퍼컴퍼니), 허위 시세 부풀리기, 중개업소 공모형, 선순위 임차인 은닉
- **삼행시 통장 사기**: 임대인 행세하며 세입자에게 "공과금·관리비 자동이체 설정" 명목으로 통장 개설·양도 유도 → 자금세탁 피해
- **오피스텔 직거래 사칭**: 집주인·중개사 동시 사칭, 소액 계약금 먼저 수령 후 잠적 — 직거래 앱 이용자 집중 피해
- **가등기 악용**: 갭투자자가 제3자에게 가등기 설정 → 본등기 시 임차인 권리 소멸 위험
- **신탁부동산 악용**: 신탁회사 동의 없이 위탁자가 임대 계약 체결 → 계약 무효, 임차인 무방비 노출 (1-6항과 연계)
- **위조 신분증/인감증명서**: 딥페이크·고품질 위조물 활용 증가 → 신분 대조만으로 식별 어려움, QR코드 진위 확인 권장

> **2026.3 업데이트**: 전세사기 누적 피해자 36,449명 (2026.3 기준), 청년(20~30대) 74.7% 집중. 삼행시 통장·오피스텔 직거래 사칭 등 신종 수법 추가 반영.

## 3. 등기부등본 구조 및 해석

### 3-1. 표제부 — 부동산 물리적 현황
- 소재지, 건물 구조, 면적, 용도, 동·호수
- **체크**: 건축물대장 면적/용도와 일치 여부

### 3-2. 갑구 — 소유권 관련

| 키워드 | 위험도 | 의미 | 앱 액션 |
|--------|--------|------|---------|
| **소유권이전** | - | 현재 소유자 확인 | 계약 상대방과 일치 확인 |
| **가압류** | 높음 | 채권자가 재산 동결, 소유자 채무 문제 | 경고 + 계약 회피 권장 |
| **가처분** | 매우 높음 | 소유권 분쟁 진행 중 | 계약 금지 |
| **경매개시결정** | 극히 높음 | 경매 절차 진행 중 | 절대 계약 금지 |
| **압류** | 매우 높음 | 세금 체납 강제집행 | 계약 금지 |
| **신탁** | 높음 | 소유권이 신탁회사에 이전 | 신탁원부 확인 + 신탁사 동의 필수 |
| **가등기** | 높음 | 소유권 이전 예약, 중간 권리 말소 가능 | 경고 |

### 3-3. 을구 — 소유권 이외의 권리

| 키워드 | 위험도 | 의미 | 앱 액션 |
|--------|--------|------|---------|
| **근저당권** | 중~높음 | 담보대출 존재 | 채권최고액 기반 위험도 계산 |
| **전세권** | 중간 | 선순위 전세 존재 | 배당 순위 확인 필요 |
| **임차권등기** | 매우 높음 | 기존 임차인 보증금 미반환 상태 | 계약 회피 강력 권장 |
| **지상권** | 중간 | 토지/건물 소유자 다를 수 있음 | 추가 확인 필요 |

### 3-4. "말소사항 포함" vs 미포함
- **말소사항 포함**: 과거 해제된 권리 이력까지 모두 표시 → 반복적 근저당/가압류 이력 = 재정 불안 시그널
- **앱 권장**: 반드시 "말소사항 포함"으로 열람 안내

## 4. 리스크 판단 지표

### 4-1. 전세가율 (전세가 / 매매가)

| 전세가율 | 등급 | 색상 | 설명 |
|----------|------|------|------|
| ~60% | 안전 | 초록 | 보증금 회수 가능성 매우 높음 |
| 60~70% | 양호 | 연두 | 일반적으로 안전한 수준 |
| 70~80% | 주의 | 노랑 | 시세 하락 시 깡통전세 위험 |
| 80~90% | 위험 | 주황 | 깡통전세 가능성 높음 |
| 90%~ | 매우 위험 | 빨강 | 계약 지양 권고 |

### 4-2. 근저당비율 계산

```
위험도 비율 = (선순위 채권최고액 + 전세보증금) / 매매시세 × 100

- 채권최고액 → 실대출 추정: 채권최고액 / 1.2
- 70% 이하: 안전
- 70~80%: 주의
- 80% 이상: 위험

예시)
  매매시세: 3억, 근저당 채권최고액: 1.2억, 전세보증금: 2억
  → (1.2 + 2) / 3 = 106.7% → 매우 위험 (깡통전세)
```

### 4-3. 임대인 신뢰도 지표

| 지표 | 위험 신호 | 확인 방법 |
|------|----------|----------|
| 세금 체납 | 국세/지방세 체납 존재 | 납세증명서 요청 |
| 소유 취득 시기 | 최근 6개월 내 취득 + 바로 전세 | 등기부등본 갑구 |
| 다수 부동산 보유 | 빌라왕 패턴 의심 | 뉴스/중개사 통한 간접 확인 |
| 법인 소유 | 소규모/최근 설립 법인 | 등기부등본 소유자 확인 |
| 근저당 변동 이력 | 잦은 설정/말소 반복 | 등기부등본 을구 (말소사항 포함) |
| 소유권 이전 빈번 | 단기간 잦은 매매 = 투기 의심 | 등기부등본 갑구 이력 |

### 4-4. 종합 위험도 스코어링 모델 (앱 적용)

```
총 위험 점수 (0~100) =
  전세가율 점수 (0~30) +
  근저당비율 점수 (0~30) +
  등기부 위험키워드 점수 (0~20) +
  임대인 신뢰도 점수 (0~20)

결과 등급:
  0~30:  안전 (초록)
  31~50: 주의 (노랑)
  51~70: 위험 (주황)
  71~100: 매우 위험 (빨강)
```

## 5. 보증금 보호 수단

### 5-1. 전세보증금반환보증보험 비교

| 항목 | HUG (주택도시보증공사) | SGI (서울보증) | HF (한국주택금융공사) |
|------|----------------------|---------------|---------------------|
| 상품명 | 전세보증금반환보증 | 전세금보장신용보험 | 전세지킴보증 |
| 가입 주체 | 임차인 | 임대인 (비용 임차인 부담 가능) | 임차인 |
| 보증금 한도 | 수도권 7억, 비수도권 5억 | 임대인 신용 기반 | 아파트 중심 |
| 전세가율 조건 | 공시가격 기준 적용 | 별도 심사 | 전세가율 기준 적용 |
| 보증료율 | 연 0.097~0.211% (4단계, 2025.3.31~) | 신용 따라 상이 | HUG 유사 |
| 보증비율 | **60%** (2025.9~ 하향) | 임대인 신용 기반 | - |
| 보증료 지원 | 최대 40만원 (2025.3.31~, 기존 30만원) | - | - |
| 특징 | 가장 보편적 | 임대인 협조 필요 | 아파트 가입 용이 |

> **2026.3 업데이트**: HUG 보증료율이 기존 3단계(0.115~0.154%) → 4단계(0.097~0.211%)로 개편(2025.3.31~). 보증비율은 기존 미기재(암묵적 70%) → **60%로 하향**(2025.9~). 보증료 지원 한도 30만원 → 40만원으로 확대. 비아파트 가입 조건: 공시가격 126% 이내(2024.5~), 향후 112% 강화 예정.

### 5-2. 확정일자 + 전입신고 법적 효력

- **대항력** (주택임대차보호법 제3조): 주택 인도(점유) + 전입신고 → **전입신고 즉시** 효력 발생 (2026.3.10 법 개정 — 기존 "다음 날 0시" 조항 폐지)
- **우선변제권** (제3조의2): 대항력 요건 + 확정일자 → 경매 시 후순위 채권자보다 우선 변제
- **핵심**: 전입신고·확정일자는 **잔금일 당일** 동시에 처리. 전출 시 대항력 상실

### 5-3. 소액임차인 최우선변제 (제8조)

| 지역 | 보증금 기준 | 최우선변제금 |
|------|-----------|-------------|
| 서울 | 1억 6,500만원 이하 | 5,500만원 |
| 수도권 과밀억제권역 | 1억 4,500만원 이하 | 4,800만원 |
| 광역시 등 | 8,500만원 이하 | 2,800만원 |
| 그 외 | 7,500만원 이하 | 2,500만원 |

> 2023년 기준. 시행령 개정에 따라 변동 가능 — 앱에 최신 기준 반영 필요

### 5-4. 전세권 설정 등기
- **장점**: 등기 공시력, 대항력(전입신고 불필요), 직접 경매 신청 가능
- **단점**: 임대인 동의 필요, 등기 비용 발생, 절차 번거로움

### 5-5. 임차권등기명령
- **요건**: 임대차 종료 후 보증금 미반환 시
- **효과**: 등기로 대항력+우선변제권 고정, 전출해도 권리 유지
- **비용**: 등록면허세 + 법원 수수료 (수만원)

## 6. 계약 단계별 체크리스트 (앱 플로우)

### Phase 1: 매물 탐색
- [ ] 국토부 실거래가로 매매가·전세가 확인
- [ ] 전세가율 계산 (70% 이하 안전)
- [ ] 주변 시세 대비 비정상적 저가/고가 경계
- [ ] 신축빌라 특히 주의
- [ ] 공인중개사 등록 여부 확인

### Phase 2: 계약 전 확인
- [ ] 등기부등본 열람 (말소사항 포함, 당일 발급)
  - 갑구: 소유자, 가압류/가처분/경매/신탁 확인
  - 을구: 근저당 채권최고액 확인
- [ ] 건축물대장 열람 (위반건축물, 용도, 면적)
- [ ] 임대인 신분증 ↔ 등기부 소유자 대조
- [ ] 납세증명서 (국세+지방세 체납 여부)
- [ ] 확정일자 부여현황 / 전입세대 확인서 (선순위 임차인)
- [ ] 전세보증보험 가입 가능 여부 사전 확인

### Phase 3: 계약 체결
- [ ] 계약서 특약: "추가 근저당 설정 금지", "보증보험 가입 협조", "잔금 전 권리변동 시 해제+위약금"
- [ ] 계약금은 소유자 본인 계좌로 송금
- [ ] 계약 당일 등기부등본 재열람
- [ ] 중개사 자격증·보증보험 확인

### Phase 4: 잔금/입주
- [ ] 잔금일 당일 등기부등본 최종 열람
- [ ] 잔금은 소유자 본인 계좌로 이체
- [ ] **당일** 전입신고 + 확정일자 + 실입주(점유) 완료
- [ ] 전세보증금반환보증 가입 신청

### Phase 5: 계약 후 관리
- [ ] 정기 등기부등본 확인 (3~6개월)
- [ ] 등기변동 알림 서비스 신청
- [ ] 만기 6개월 전 갱신/반환 의사 통보
- [ ] 미반환 시 임차권등기명령 신청 준비

### 7-1. 국토부 실거래가 API (핵심)

| API | 용도 |

| `getRTMSDataSvcAptTradeDev` | 아파트 매매 실거래가 |
| `getRTMSDataSvcAptRent` | 아파트 전월세 실거래가 |
| `getRTMSDataSvcRHTradeDev` | 연립다세대 매매 실거래가 |

| `getRTMSDataSvcSHRent` | 단독/다가구 전월세 실거래가 |

- **파라미터**: `LAWD_CD`(법정동 5자리) + `DEAL_YMD`(YYYYMM) + `serviceKey`
- **무료**, 공공데이터포털 인증키 발급

### 7-2. 건축물대장 API

- **무료**, 공공데이터포털

### 7-4. API 접근 불가 항목 (앱에서 가이드 제공)
- **등기부등본**: 대법원 인터넷등기소 — 공개 API 없음 (건당 700원 유료 열람). → 앱에서는 **등기부등본 텍스트를 사용자가 붙여넣기** → Claude가 해석하는 방식
- **납세증명서**: 홈택스/위택스 직접 발급 → 앱에서 안내 링크 제공
- **전입세대 확인서**: 주민센터 발급 → 체크리스트 항목으로 안내
