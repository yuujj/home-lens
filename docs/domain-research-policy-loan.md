# 한국 정책 대출 도메인 리서치

## Context

해커톤 앱 개발을 위해 한국 주요 정책 대출 상품의 자격 조건, 금리 구조, LTV/DTI/DSR 규제, 대출 한도 계산 로직, 활용 가능한 API를 체계적으로 정리한다. 앱 개발팀이 사용자 입력 → 가능한 상품 필터링 → 예상 한도/금리 계산 로직을 바로 구현할 수 있는 수준을 목표로 한다.

**조사 기준일**: 2025년 기준 (금리는 변동 가능, 상품 조건은 국토교통부 고시 기준)

---

### 1-2. 디딤돌대출 (내집마련 구입자금)

**운영기관**: 주택도시기금 (nhuf.molit.go.kr)
**취급은행**: 우리·신한·KB국민·NH농협·KEB하나·대구·부산은행

#### 자격 조건

| 항목 | 일반 | 생애최초 | 신혼가구 | 2자녀 이상 |
|------|------|---------|---------|-----------|
| 연소득 (부부합산) | 6천만원 이하 | 7천만원 이하 | 8,500만원 이하 | 7천만원 이하 |
| 순자산 | 5.11억원 이하 | 5.11억원 이하 | 5.11억원 이하 | 5.11억원 이하 |
| 주택 보유 | 무주택 세대주 | 생애최초 무주택 | 무주택 세대주 | 무주택 세대주 |
| 연령 | 세대주 (만 30세 미만 단독세대주 제외) | 제한 없음 | 혼인 7년 이내 | 제한 없음 |

> **순자산 기준**: 통계청 가계금융복지조사 소득 4분위 전체가구 평균값 (매년 변동, 2025년 기준 5.11억원)

#### 주택 조건

| 항목 | 일반 | 신혼·2자녀 이상 | 만 30세 미만 단독세대주 |
|------|------|--------------|---------------------|
| 주택가격 | 5억원 이하 | 6억원 이하 | 3억원 이하 |
| 전용면적 | 85㎡ 이하 | 85㎡ 이하 | 60㎡ 이하 |
| 읍·면 지역 | 100㎡ 이하 | 100㎡ 이하 | 70㎡ 이하 |
| 주택 유형 | 아파트·단독·다가구·연립·다세대 | 동일 | 동일 |

| 생애최초 | 3억원 이하 | 80% (수도권·규제지역 70%) |
| 신혼·2자녀 이상 | 4억원 이하 | 70% |
| DTI | 60% 이내 | - |

#### 금리표 (2025년 기준, 변동금리)

| 2~4천만원 | 2.75% | 2.85% | 2.95% | 3.05% |
| 4~6천만원 | 3.05% | 3.15% | 3.25% | 3.35% |

> 금리는 국토교통부 고시에 따라 변동. 최신 금리는 nhuf.molit.go.kr 또는 hf.go.kr에서 확인 필요.

| 저소득 청년 (만 40세 미만) | -0.1%p |
| **최대 우대** | **-1.0%p** |

## 2. LTV / DTI / DSR 규제

### 2-1. 지표 정의 및 계산 공식

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
LTV (Loan To Value ratio, 주택담보인정비율)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
LTV = 대출금액 / 주택 담보가치 × 100

예시) 주택가격 5억원, 대출 3억원
    LTV = 3억 / 5억 × 100 = 60%

→ 대출 가능 최대 금액 = 주택 담보가치 × LTV 한도
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
DTI (Debt To Income ratio, 총부채상환비율)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
DTI = (주택담보대출 연간 원리금 상환액 + 기타 대출 연간 이자) / 연소득 × 100

예시) 연소득 6천만원, 주담대 원리금 2천만원, 기타대출 이자 3백만원
    DTI = (2,000 + 300) / 6,000 × 100 = 38.3%

→ 대출 한도 = (연소득 × DTI 한도 - 기타대출 이자) / 연간 원리금률
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
DSR (Debt Service Ratio, 총부채원리금상환비율)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
DSR = 모든 대출의 연간 원리금 상환액 / 연소득 × 100
    = (주택담보대출 원리금 + 신용대출 원리금 + 자동차할부 원리금 + ...) / 연소득 × 100

DTI와 차이점:
  DTI: 기타대출은 이자만 포함
  DSR: 모든 대출의 원금+이자 포함 (더 엄격)

→ 총 대출 1억원 초과 시 은행권 DSR 40%, 비은행권 50% 적용
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 2-5. 스트레스 DSR (2025년 7월 3단계 시행)

```
스트레스 DSR = (모든 대출 원리금 + 스트레스 금리 가산분) / 연소득 × 100

스트레스 금리: 과거 5년간 최고금리 - 현재금리 (최소 0.75%p, 최대 3.0%p)

3단계(2025.7~): 전국 전면 적용
  1단계(2024.2~): 은행권 주담대 0.38%p
  2단계(2024.9~): 은행권 전체·2금융 주담대 0.75%p
  3단계(2025.7~): 전금융권 모든 대출 적용

→ 실질 대출 가능 금액이 스트레스 DSR 적용으로 축소됨
→ 디딤돌·버팀목 등 정책 대출은 스트레스 DSR 제외 (별도 DTI 기준 적용)
```

## 4. 대출 한도 계산 로직

### 4-1. 한도 계산 공식

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
디딤돌대출 한도 계산
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
최종 한도 = MIN(
  ① 주택가격 × LTV,
  ② 연소득 기반 DTI 한도,
  ③ 상품별 최대 한도 (2.5억 / 3억 / 4억)
)

② DTI 한도 = (연소득 × 0.6 - 기타대출 연간이자) / 연간원리금계수
   연간원리금계수 = 월 원리금 × 12 / 대출금액 (만기·금리에 따라 결정)

예시 계산)
  연소득: 5천만원, 주택가격: 4억원, 만기: 30년, 금리: 3.0%
  ① LTV 70% 기준: 4억 × 70% = 2.8억
  ② DTI 60%: 5천만 × 60% = 3천만원 가용 원리금
     30년 3.0% 월 원리금계수: 약 0.00421
     한도 ≒ 3천만원 / (0.00421 × 12) ≒ 5.93억 → 상한 초과
  ③ 상품 최대: 2.5억원
  → 최종 한도: MIN(2.8억, 5.93억, 2.5억) = 2.5억원

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
버팀목전세대출 한도 계산
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
최종 한도 = MIN(
  ① 전세보증금 × LTV (80%),
  ② 상품별 최대 한도 (1.2억 / 2억 / 2.4억)
)

예시 계산)
  전세보증금: 2억원, 일반 버팀목
  ① 2억 × 80% = 1.6억
  ② 일반 최대: 1.2억원
  → 최종 한도: MIN(1.6억, 1.2억) = 1.2억원

예시 계산 (신혼가구))
  전세보증금: 2.5억원, 신혼 버팀목
  ① 2.5억 × 80% = 2억
  ② 신혼 최대: 2억원
  → 최종 한도: MIN(2억, 2억) = 2억원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 연소득 | 디딤돌 (일반) | 디딤돌 (신혼) | 버팀목 (일반) | 버팀목 (신혼) | 보금자리론 |
|--------|------------|------------|------------|------------|---------|

| 4~6천만원 | 2.5억 | 4억 | 1.2억 | 2억 | 3.6억 |
| 6~7천만원 | 불가 (초과) | 불가 | 불가 (초과) | 불가 | 3.6억 |
| 7~8.5천만원 | 불가 | 4억 (신혼) | 불가 | 2억 (신혼) | 3.6억 (신혼) |

### 4-3. 실제 계산 예시

```
[예시 1] 연소득 4천만원, 신혼부부, 주택가격 4.5억원 (일반지역)

1단계: 가능 상품 확인
  - 디딤돌 신혼: 소득 8.5천만원 ↓ ✓, 주택가격 6억원 ↓ ✓
  - 보금자리론: 소득 8.5천만원 ↓ ✓, 주택가격 6억원 ↓ ✓

2단계: 디딤돌 신혼 한도 계산
  - LTV 70%: 4.5억 × 70% = 3.15억
  - 상품 최대: 4억
  - DTI 60%: (4천만 × 60%) / 연간원리금계수 ≒ 여유 있음
  → 한도: MIN(3.15억, 4억) = 3.15억원

3단계: 보금자리론 한도 계산
  - LTV 70%: 4.5억 × 70% = 3.15억
  - 상품 최대: 3.6억 (신혼 8.5천 ↓)
  → 한도: MIN(3.15억, 3.6억) = 3.15억원

4단계: 금리 비교
  - 디딤돌 신혼: 약 2.75~3.05% (변동) + 신혼 -0.2%p
  - 보금자리론: 4.0~4.1% (고정) + 신혼 -0.3%p = 3.7~3.8%
  → 디딤돌이 금리 낮음, 단 변동 위험 존재

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[예시 2] 연소득 1억원, 맞벌이, 출산 1년 (신생아), 주택 7억원 (일반지역)

1단계: 가능 상품 확인
  - 디딤돌: 소득 6천만원 초과 → 불가 ✗
  - 보금자리론: 소득 1억 이하 (다자녀 기준) → 주택 6억 초과 → 불가 ✗
  - 신생아특례 구입: 소득 1.3억 이하 ✓, 주택 9억 이하 ✓

2단계: 신생아특례 한도 계산
  - LTV 70%: 7억 × 70% = 4.9억
  - 상품 최대: 5억
  → 한도: MIN(4.9억, 5억) = 4.9억원

3단계: 특례 금리 적용
  - 연소득 1억 → 연 3.3% 내외 (특례 5년)
  - 5년 후 가산금리 전환 리스크 고지 필요

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[예시 3] 연소득 3천5백만원, 만 29세 청년, 전세 1.8억원 (서울)

1단계: 가능 상품 확인
  - 청년전용 버팀목: 연령 ✓, 소득 5천만원 ↓ ✓, 보증금 3억 ↓ ✓
  - 일반 버팀목: 소득 ✓, 보증금 3억 ↓ ✓ (하지만 청년전용이 금리 우위)

2단계: 청년전용 버팀목 한도 계산
  - LTV 80%: 1.8억 × 80% = 1.44억
  - 상품 최대: 2억
  → 한도: MIN(1.44억, 2억) = 1.44억원

3단계: 금리
  - 연소득 구간 2~4천만원 → 약 1.3~2.0%대 (우대 적용 전)
```

### 5-3. 자격 판단 핵심 필터링 로직

```
[Step 1] 용도 필터
  구입 목적 → 디딤돌·신생아특례구입·청년주택드림·보금자리론 검토
  전세 목적 → 버팀목·청년버팀목·신생아특례전세 검토

[Step 2] 소득 필터
  각 상품별 소득 상한 체크
  맞벌이 여부에 따라 합산 소득 또는 개인 소득 기준 적용

[Step 3] 자산 필터
  순자산 = 총자산 - 총부채
  상품별 순자산 상한 체크

[Step 4] 주택 조건 필터
  구입가 or 전세금이 상품 한도 이내인지 확인
  전용면적 85㎡ 이내 확인 (읍·면 100㎡)

[Step 5] 특수 조건 필터
  청년버팀목: 만 19~34세 확인
  청년주택드림: 만 20~39세 + 청약 당첨 필수
  신생아특례: 2년 이내 출산/입양 확인
  신혼: 혼인 7년 이내 확인
  생애최초: 과거 주택 소유 이력 없음 확인

[Step 6] 한도 계산
  통과 상품 각각에 대해
  MIN(주택가(전세금) × LTV, 상품최대한도) 계산

[Step 7] 금리 계산
  기본 금리 (소득 구간 기반)
  - 우대금리 항목 체크 (신혼·자녀·청약 등)
  = 최종 예상 금리 범위 출력

[Step 8] 결과 정렬
  금리 낮은 순으로 추천 우선순위 정렬
  한도 부족 시 '부분 가능' 표시
```

### 7-3. UX 설계 고려사항

```
[입력 단계 설계 권장]

Step 1: 기본 정보 (3개)
  → 대출 목적 / 연소득 / 주택 가격(전세금)
  → 이 3개만으로 1차 가능 상품군 즉시 표시 (진행 동기 부여)

Step 2: 자격 세부 확인 (5개)
  → 나이 / 혼인 여부·기간 / 주택 보유 이력 / 소재지 / 면적

Step 3: 우대 항목 (선택)
  → 자녀 수 / 신생아 여부 / 청약통장 등
  → "더 정확한 금리를 위한 추가 정보" 표현 권장

[UX 주의사항]
- 순자산 입력: 계산 방법 툴팁 필수 ("총자산 - 총부채")
- 연소득 입력: 세전/세후 혼란 방지 → "세전 연봉 또는 종합소득세 신고 기준" 안내
- 혼합 소득(근로+사업): 합산 방법 안내
- 결과 화면: 금리는 "예상 범위"로 표시, "실제 금리는 금융기관 심사에 따라 상이" 고지
- 복수 상품 추천 시: 금리 낮은 순 + 한도 큰 순 정렬, 비교표 제공
```

## 8. 법적 고려사항

- 모든 계산 결과에 "본 결과는 참고용이며 실제 대출 가능 여부는 금융기관 심사 결과에 따라 다를 수 있습니다" 면책 고지 필수
- 금리·조건은 국토교통부 고시에 따라 변동 가능 — 최신 기준 확인 링크 제공 권장
- 대출 중개 행위로 오해받지 않도록 "정보 제공 서비스"임을 명시
- 개인정보(소득·자산) 입력 시 저장 여부 및 처리 방침 안내
- 공공데이터 활용 시 공공데이터 이용조건(공공데이터법) 준수
